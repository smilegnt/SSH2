光盘说明


一、版权说明

    本光盘为《Struts 2 + Spring 3 + Hibernate框架技术精讲与整合案例》一书的配书光盘，所有内容仅供读者配合教材学习和练习使用。未经许可，不得以任何理由复制散发或用于其他商业目的，违者必究。


二、光盘内容

1. 代码  

   本书所涉及实例的所有代码均包含在光盘内，代码的组织结构与书的组织结构一致，按章节划分，有名称的按名称命名，无名称的按序号排列。

2. 项目部署说明
  
   在每一篇的文件夹内，有一个项目部署说明文件，是对环境搭建及相关变量配置的说明。读者在程序调试遇到问题时可参阅此文件，以确保关键参数配置正确无误。

   在综合实例篇，每个实例配有一个专门的项目部署说明文件，对特殊参数配置进行了说明和讲解。

3. 各项目所用JAR包

   为相应章节的程序在调试中需要添加的JAR包。如果读者的调试环境中未包含该JAR包，可参照书中的方法将JAR包加入调试环境。

4. 数据库文件

   为MySQL数据库部署文件，读者可以在MySQL下使用该文件生成初始数据库。


三、运行环境

1. 硬件配置

   主流PC机，奔4以上CPU，1GB以上内存，有线或无线网络连接。

2. 软件环境

   Windows XP及以上操作系统；Java 7.0或更高版本，Tomcat 7.0以上版本，MySQL 5.5以上版本。
